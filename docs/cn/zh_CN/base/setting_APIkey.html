API Key

配置API Key

进入编程模式

开机后，在主菜单选择编程模式,等待信号指示灯右红变成绿色（这表示M5GO连接网络成功），用平板扫描屏幕上的二维码，或是在电脑浏览器直接访问flow.m5stack.com进入UIFlow编程页面

填写API Key

进入UIFlow后，点击页面右上角的菜单栏中的设置按钮，输入对应M5GO上的API Key，点击OK保存，等待提示连接成功
