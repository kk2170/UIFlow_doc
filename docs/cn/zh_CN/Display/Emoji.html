Emoji

Emoji

功能说明

UIFlow提供可以快速绘制图案的像素点编辑Block，通过拼凑像素点的形式可以绘制各种各样的图案

  • Set emoji map in 设置方阵像素点，勾选方阵中的小方格对应显示对应的像素点
  • Set line x row x in 设置单个像素点，选择行与列，对应显示对应的单个像素点

使用方法

将Emoji Block添加到程序中，勾选小方格激活像素点
